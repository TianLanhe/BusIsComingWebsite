# Homepage UI Optimization Design

**日期**：2026-06-16
**状态**：用户已确认
**范围**：优化 BusIsComing 网站首页下载入口、首屏功能展示、截图素材、本地化文案、城巴产品口径和服务监听入口。

## 背景

当前首页已经具备 Android APK 下载、在线查询静态演示、功能介绍、下载区和 FAQ。用户反馈现有下载按钮不符合使用习惯，并进一步提出首屏功能展示、真实截图、香港繁中文案、服务监听地址和城巴范围口径的优化需求。

本设计不重开 Android APK 下载后端业务逻辑，也不修改现有下载 API 路径。重点是首页体验、内容口径、视觉素材和运行入口配置。

## 已确认决策

1. 首屏主按钮改为「下載 Android APK」，点击直接触发 Android APK 下载，不再跳转到下载区。
2. 首屏移除 Android/iPhone 大平台框，只保留 `APK 1.0 · 約 4.8 MB` 和低权重「iPhone 暫未支援」。
3. 下载区改为 Android 详情卡，作为备用下载入口和平台状态确认。
4. 下载失败在按钮附近原位提示；普通用户界面不展示完整 SHA-256。
5. 首屏右侧展示模块按 4 个核心功能点自动轮播，每 4.5 秒切换，动画 600-700ms，无左右箭头。
6. 自动轮播只发生在 4 个核心功能点之间；同一功能点内的多张截图只允许用户手动点选切换。
7. 同一功能点内的截图采用堆叠图集：主图完整展示，底层截图露出边缘；点选底层截图后升为主图。
8. 展示模块桌面端内部采用左图右文；手机端采用图上文下。
9. 真实截图不再套手机边框，只保留轻圆角、细边框和柔和阴影。
10. 截图按 4 个核心展示点分组，序号 `1` 的图片作为主图；单图功能点不显示堆叠切换。
11. 前端只使用脱敏副本。地点、站名、路线号、搜索记录、系统无关区域需要处理；价格、时间、ETA 保留。
12. 网站产品口径从「香港巴士查询」收窄为「香港城巴 / Citybus 查询」。
13. 范围说明必须明确不提供九巴、港铁、铁路、渡轮或其他交通工具查询。
14. 6 个功能卡片中的「HK$ 清晰顯示」改为强调「多程全程總車費」。
15. `zh-Hant` 是独立香港文案，不是 `zh-Hans` 的繁体映射。繁中应逐句调整语气，使用香港书面语偏自然的表达。
16. 前后端开发入口监听所有本机网络接口，支持同一局域网设备访问。
17. 前端正式部署支持标准 HTTP 入口；端口通过环境配置，非 root 环境可通过反向代理或端口映射提供。

## 视觉结构

### 首屏下载入口

首屏保留两个主要操作：下载 Android APK 和在线查询。下载按钮承担真实下载动作，在线查询仍为次操作。按钮下方只保留短元信息和 iPhone 状态，避免平台分段控件抢占主转化。

### 下载区

下载区成为确认和备用入口，展示 Android APK 版本、大小、下载按钮和 iPhone 暂未支持状态。它不展示完整 SHA-256，也不提供历史版本选择。

### 功能展示模块

展示模块有两层状态：

- 外层功能轮播：4 个核心功能点自动切换，使用深度换层动画。
- 内层截图图集：每个功能点的多张真实截图手动切换，使用堆叠视觉表达。

外层自动轮播需要在用户 hover、focus、点按截图堆叠时暂停，避免自动切换打断用户查看截图。

## 内容口径

首页所有查询相关文案优先使用「城巴」或 `Citybus`。可保留「香港」作为地域上下文，但不能暗示覆盖九巴或所有香港巴士公司。

繁中应按香港用户习惯独立写作。例如「網上查詢」「一撳」「總車費」「暫未支援」等表达可以用于合适场景，但整体仍保持清楚、可信、不过度口语化。

## 截图处理

截图资产统一沉淀在 `frontend/src/assets/app-screenshots/real/`，前端页面不直接引用项目外原图。实现阶段需要生成或确认可展示的前端资产副本。

脱敏规则：

- 替换真实地点、站名、路线号和搜索记录。
- 模糊或裁掉手机状态栏、系统导航、通知栏、账号、输入法、运营商等与 App 无关的信息。
- 保留价格、时间和 ETA 数值。
- 保留真实 App UI、布局、状态和数据形态。

## 验证重点

- 首屏点击「下載 Android APK」直接开始下载，不发生锚点跳转。
- iPhone 状态在首屏和下载区都清楚但低权重。
- 功能展示自动轮播无箭头，4 个核心功能点顺序清晰。
- 同功能截图堆叠只能手动切换，不能自动播放。
- 桌面和手机布局都能清楚查看主图、底层截图和说明文案。
- `zh-Hant` 关键文案不是 `zh-Hans` 的字形转换。
- 页面不再泛称香港巴士查询，必须明确当前聚焦城巴。
- 开发者能从同一局域网设备访问本机前后端服务。
