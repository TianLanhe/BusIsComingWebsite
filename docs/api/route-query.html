在线路线查询 API

本页是路线查询三接口的中文静态预览。权威契约为 shared/contracts/openapi/route-query-api.openapi.yaml。

接口

  • POST /api/routes/query_places：地点检索，返回候选地点和 placeToken。
  • POST /api/routes/query_routes：路线摘要查询，只接收起终点 token，最多返回 20 条香港巴士路线。
  • POST /api/routes/query_etas：批量 ETA 查询，返回每条路线的首程候车状态。

Envelope

{
  "requestId": "route-query-example",
  "data": {},
  "error": null
}

验证命令

cd frontend
npm run openapi:routes:lint
npm run openapi:routes:bundle
