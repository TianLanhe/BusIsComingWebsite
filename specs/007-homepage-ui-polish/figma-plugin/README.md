# 首页 UI 体验优化补充 Figma 插件

本插件用于在目标 Figma 文件中生成 `Homepage UI Polish - 007` 页面和关键设计节点。

## 使用方法

1. 打开 Figma 文件 `BusIsComing Website - Homepage v1 Spec`。
2. 选择 `Plugins -> Development -> Import plugin from manifest...`。
3. 选择本目录下的 `manifest.json`。
4. 运行插件。
5. 复制插件输出的节点报告，并回填到 `specs/007-homepage-ui-polish/figma.md` 或后续任务记录。

## 生成内容

Figma file: https://www.figma.com/design/LAm6RjzFuFHsHFlcipx8pU/BusIsComing-Website---Homepage-v1-Spec

Page: `Homepage UI Polish - 007`

- `Desktop 1440 / Hero Medium Screenshot Deck`
- `Desktop 1440 / Screenshot Lightbox`
- `Mobile 390 / Compact Feature Grid`
- `Mobile 390 / Compact Route Result Card`
- `Interaction States / Split Gesture Zones`
- `Spec Notes`

插件是计划阶段 Figma MCP 需要重新认证时的 fallback；运行后仍需要人工检查视觉效果。
