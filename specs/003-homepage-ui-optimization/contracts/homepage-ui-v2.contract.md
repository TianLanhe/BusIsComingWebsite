# UI 契约：首页 v2 交互与状态

## 适用范围

本契约覆盖首页首屏下载入口、下载区详情卡、首屏功能展示、截图图集、三语状态保持、响应式布局和本地/正式访问入口。实现可以调整组件命名，但不得改变这些用户可见状态和不变量。

## 首屏下载入口

| 项 | 契约 |
|----|------|
| 主按钮行为 | 首屏主按钮直接进入 Android APK 下载路径 `/api/downloads/android/latest`，不滚动到下载区，不展开 Android/iPhone 大分段框 |
| 标签 | 三语表达为下载 Android APK；`zh-Hant` 必须自然香港书面语 |
| 元信息 | 只显示简短 APK 版本和约 4.8 MB；普通用户界面不展示完整 SHA-256 |
| iPhone 状态 | 低权重展示暂未支持，不提供下载路径，不伪装失败 |
| 失败提示 | 下载失败或暂不可用时在按钮附近原位展示三语提示，不导向 iPhone 或其他平台 |

## 下载区详情卡

| 项 | 契约 |
|----|------|
| Android 卡片 | 展示版本、大小、主要下载入口和备用下载入口 |
| iPhone 状态 | 作为状态说明出现，不与 Android 入口竞争视觉优先级 |
| 行为一致性 | 下载区与首屏使用同一 Android 下载路径和同一失败语义 |

## 功能展示外层自动轮播

| 项 | 契约 |
|----|------|
| 项数 | 必须为 4 个核心功能点 |
| 顺序 | 常用城巴路线快速查询、路线比较、多班 ETA / 路线详情、出门前监控与语音提醒 |
| 自动切换 | 默认约 4.5 秒切换到下一项；10 秒内至少完成 1 次切换 |
| 动画 | 使用 600-700ms 深度换层/层级切换动画，不使用装饰性光斑或夸张营销动效 |
| 控件 | 不显示左右箭头；可使用非箭头式当前项指示 |
| 暂停 | hover、focus、pointer/touch 交互期间暂停；离开或结束交互后恢复 |
| 减少动态效果 | `prefers-reduced-motion: reduce` 下停止自动动画或降级为无位移动画 |

## 同功能点截图图集

| 项 | 契约 |
|----|------|
| 主图 | 序号或文件名为 `1` 的脱敏截图默认作为主图 |
| 堆叠 | 多图功能点露出底层截图边缘，用户可手动选择 |
| 手动切换 | 用户选择底层截图后，该图升为主图，原主图退回堆叠层 |
| 禁止项 | 同组截图不得自动轮播，不得提供左右箭头 |
| 单图功能点 | 只显示主图，不显示空堆叠目标 |
| 视觉边界 | 不套手机边框；使用克制圆角、描边或阴影与页面背景区分 |

## 截图资产脱敏

- 前端只引用 `frontend/src/assets/app-screenshots/real/` 下的项目内可展示资产，不引用项目外原图。
- 必须遮蔽或替换真实地点、站名、路线号、搜索记录和手机系统无关内容。
- 必须保留价格、时间和 ETA 数值。
- 替代文本不得包含真实地点、真实站名或真实路线号。

## 文案与范围

- 所有用户可见文字覆盖 `zh-Hant`、`zh-Hans`、`en`。
- `zh-Hant` 独立编写，不能由 `zh-Hans` 单纯转换。
- 页面提到查询范围时必须聚焦 Citybus / 城巴。
- 页面必须说明当前不提供九巴、港铁、铁路、渡轮、其他交通工具查询或完整出行规划。
- 6 个功能卡片中的价格卡片必须强调“多程全程总车费”，而不是只强调 HK$ 币种。

## 状态保持

| 场景 | 契约 |
|------|------|
| 切换语言 | 保留当前核心功能点和当前主图 |
| 外层自动切换 | 只更新核心功能点，不改变其他功能点的用户手动主图选择 |
| 用户手动选图 | 暂停外层自动轮播，避免切换过程中打断用户 |
| 下载失败 | 不改变 iPhone 状态，不切换到其他平台 |

## 响应式布局

| 视口 | 契约 |
|------|------|
| 桌面 1440px | 展示区左图右文；下载按钮、在线查询按钮、APK 元信息、iPhone 状态完整可读 |
| 手机 390px | 展示区图上文下；堆叠露出区域可触控；主要按钮和文案不重叠、不截断 |
| 窄屏 | 关键操作可见可操作，文字必要时换行，不使用 viewport 宽度直接缩放字体 |

## 访问入口

- 前端开发和预览必须支持监听所有本机网络接口，供同局域网设备访问。
- 后端下载服务必须支持监听所有本机网络接口，供同局域网前端访问下载 API。
- 正式前端可通过服务自身、反向代理或端口映射提供标准 HTTP 入口。
- 监听配置调整不得改变下载 API 路径、响应头或错误语义。
