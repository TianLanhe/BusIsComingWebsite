# 首页体验精修 Figma 插件

本插件用于在目标 Figma 文件中生成 `Homepage Experience Polish - 005` 页面和关键设计节点。

## 使用方法

1. 打开 Figma 文件 `BusIsComing Website - Homepage v1 Spec`。
2. 选择 `Plugins -> Development -> Import plugin from manifest...`。
3. 选择本目录下的 `manifest.json`。
4. 运行插件。
5. 复制插件输出的节点报告，并回填到 `specs/005-homepage-experience-polish/figma.md` 或后续任务记录。
