# 繁体文案审校：首页体验精修

## 审校范围

- Header、hero、功能轮播、功能卡、网路试查区、下载区、FAQ、footer、按钮、状态提示、图片 alt 和 aria 文案。
- `zh-Hant` 独立按香港实用书面语撰写；`zh-Hans` 和 `en` 保持同一产品事实，但不硬套香港繁体表达。

## 参考来源

- Citybus: `https://www.citybus.com.hk/`
- HKeMobility: `https://www.hkemobility.gov.hk/`
- Transport Department: `https://www.td.gov.hk/`
- GovHK public transport: `https://www.gov.hk/tc/residents/transport/publictransport/`

以上来源只作为香港交通产品措辞参考；App 功能事实仍以 Android 主项目和本 feature 规格为准。

## 关键用词

| 概念 | `zh-Hant` 采用 | 说明 |
|------|----------------|------|
| online query | 網上試查 | 比「在線查詢」更贴近香港网站语境，并避免承诺完整服务 |
| ETA / arrival | 抵站時間 | 交通产品语境下比直译 ETA 更自然 |
| fare | 交通費用 | 比「总车费」更适合概括路线比较 |
| saved routes | 儲低 / 儲存常用路線 | 保持清楚、实用，不采用过度口语 |
| contact | 聯絡我們 | 替代旧的「支援我們」，避免捐助或支持语义 |

## 范围边界

- 当前聚焦 Citybus / 城巴路线查询。
- 不提供九巴、港铁、铁路、渡轮、其他非城巴交通工具查询。
- 不提供完整出行路线规划。
- 网页只提供基础城巴路线试查；保存路线、监测、多班抵站时间和更多详情仍引导下载 App。

## 审校结论

- 关键 `zh-Hant` 文案已避免从 `zh-Hans` 直接字形转换。
- 交通相关描述统一使用「抵站時間」「交通費用」「網上試查」「聯絡我們」等香港语境表达。
- 英文与简体中文保留同一产品事实和范围限制，不强行使用香港繁体词形。
