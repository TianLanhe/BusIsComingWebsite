# Figma 对照与浏览器截图验收清单

## Figma 来源

- 文件：https://www.figma.com/design/LAm6RjzFuFHsHFlcipx8pU
- 桌面首页：node `4:2`
- 手机首页：node `4:183`

- 轮播状态：node `4:357`

## 验收清单

- [X] 桌面首屏信息层级、CTA 优先级和左右布局与 node `4:2` 一致。
- [X] 手机首屏内容顺序、单列节奏和 CTA 可见性与 node `4:183` 一致。
- [X] 下载按钮 default、Android 展开、iPhone 展开状态与 node `4:326` 语义一致。
- [X] 轮播 4 项顺序、分页状态和说明区域与 node `4:357` 一致。
- [X] 在线查询区保持静态演示，不表现为完整实时查询。
- [X] 核心文字无重叠，主要按钮未截断，App 预览未遮挡文字。
- [X] 页面在 1440px 桌面和 390px 手机 viewport 均无横向滚动。

## 已知偏差

- 当前 Android 主项目未提供真实页面截图，网站使用临时高保真 mock，并在页面和内容模型中标注待替换。
- 当前网站仓库未提供正式 APK，Android 下载区域显示“下载资源待接入”的可信降级状态，不伪造可安装包。
