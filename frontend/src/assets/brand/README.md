# Brand Assets

- `busiscoming-logo-foreground.png` 来自 Android 主项目 `/Users/jianglijie/AndroidStudioProjects/BusIsComming/app/src/main/res/mipmap-xxxhdpi/ic_launcher_foreground.png`。
- 导出时按 alpha 非透明像素裁切，保留 18px 安全边距，再输出为 512x512 PNG 透明背景。
- `frontend/public/favicon.webp` 基于同源 foreground 图输出为 96x96 WebP。
- Header、footer 和 favicon 必须使用这个 foreground 口径，不使用 launcher 背景底板、渐变底色、圆角底板或 lucide 线框巴士。

旧的 `busiscoming-icon.webp` 是完整 launcher 图标，仅保留作历史素材，不再作为网站品牌 logo。
